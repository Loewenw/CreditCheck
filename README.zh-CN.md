# Ant Design Pro

基于Ant Design Pro实现的信审后台系统。


## 模板

```
- 用户信息查询
  - 总览
  - 账单
  - 淘宝
  - 运营商通讯录
  - 网查
  - 审计结果
- 帐户
  - 登录
  - 注册
  - 注册成功
```

## 使用

```bash
$ git clone https://github.com/Loewenw/CreditCheck.git --depth=1
$ cd CreditCheck
$ npm install
$ npm start         # 访问 http://localhost:8000
```

更多信息请参考 [使用文档](http://pro.ant.design/docs/getting-started)。

## 兼容性

现代浏览器及 IE11。
